# Ice Cream Ordering System (MS Access)

This project is a database-based Ice Cream Ordering System developed using Microsoft Access. It helps manage customer orders, flavors, toppings, and billing efficiently.

##Features
- Manage customer information
- Store ice cream flavors and prices
- Add toppings with pricing
- Place and track orders
- Generate total bill
- Create reports for orders

##Database Structure

### Tables
- **Customer**
  - Name
  - Phone
  - Address

- **Flavor**
  - Flavor Name
  - Price

- **Toppings**
  - Topping Name
  - Topping Price

- **Orders**
  - Customer ID
  - Flavor ID
  - Topping ID
  - Quantity
  - Order Date
  - Total Amount

##Queries
-Bill Query
  - Calculates total bill for customer orders

##Forms
- Customer Form
- Order Form
- Bill Query Form

##Reports
- Ice Cream Order Report

##How to Use
1. Open the project in Microsoft Access
2. Enter customer data
3. Select flavors and toppings
4. Place orders using the form
5. View bill and reports
